如何有效学习

今天重新温习了自己在这一年来所看到的好的学习方法。
特意整理出来，留作自己和有兴趣的读者日后的参考。希望大家能够从中学到好的知识。

学习和阅读

我们平日里学习新知识的最主要方式，就是阅读。
所以，掌握有效的阅读的方法，是进行有效学习的一个关键。
目前，我看到的比较好的一个读书方法是万维钢老师提出的 “强力研读”。

根据万老师的说法，强力研读最核心的是做好 读书笔记。

不过，我有一个疑更加接近爱因斯坦的思维方式呢？惑是，如何做好读书笔记呢？

以下是万老师的说法：

读书笔记首先要清晰表现每一章的逻辑脉络。它第一作用就是抛开故事记住文章。让一本书从厚变薄，从具体的山川景色变成抽象的地图。只有当你跳出字里行间，以居高临下的姿态俯视全章，它的脉络才能变得清晰。看清楚以后不要抄作者的话，用自己的语言把这个脉络写出来，就好像画地图一样。
逻辑脉络之外，如果发现真正好的小故事 —— 我们称之为”亮点”—— 那么就把这个故事也给写下来，甚至具体到细节。读书，在某种程度上就是寻找能够刺激自己思维的那些亮点。我们在分析脉络的时候要忽略故事，分析完脉络再把故事带走

以上可以说是做好读书笔记的一个 总体战术。

不过，我觉得以上的描述只告诉了我们好的学习笔记应该长成什么样子，但是，并没有告诉我们如何做出这样的笔记。

我认为要做出好的学习笔记，首先要明白什么样算是好。

好是一个非常主观的概念。也就是说，没有最好，只有更好。先来思考一个问题，假设你和爱因斯坦同时学习一个最前沿的物理概念。你们两个谁能够对这个新的物理概念产生更加 好 的洞见或者想法？

这个答案比较明显，大部分情况下，爱因斯坦显然是要比你厉害不少的。

但是这个是为什么呢？为什么爱因斯坦的想法，比你更好呢？
一个重要的原因是，爱因斯坦对这个概念理解的比你更加深刻。

为什么爱因斯坦比你理解的更加深刻？

排除掉智力因素，一个关键的因素是：爱因斯坦和你思考问题的角度，考虑问题的方式，是不在一个层级的。

那么，这又是为什么呢？或者说，如何能够让你的思维模式到达不同的层级呢?

显然，造成这个差距的因素很多，不过其中应该还是有几个重要的点。

  1. 你们的知识储备不同。
  2. 你们的思考方式不同。

知识储备，指的是你们真正的理解了的知识量，而不是知道的知识量。

思考方式，是你们如何理解一个知识。

可以说，不同的思考方式造成了对一个问题理解程度深浅的区别。

而对一个问题理解深浅的不同，造成了你们知识上储备的不同。

因此，最关键的一点还是掌握好的思维方式。

我在和教授做研究的过程中，对教授们的思考方法有一些观察。我发现，教授在平日的讨论中，很少问非常具体的细节。他常常说的一句话是，“I need to understand this conceptually first”。

在这个过程中，他会把一个新的概念，和自己以往学习到的知识尝试做一个连接。并且尝试在自己的脑中建立一个初步的印象。在完成初步概念构建后，我们在后续的会议中，才会具体的讨论实现的细节。

这和冯诺依曼的“自顶向下”的学习方法有着很大的相似性。也就是，先制定一个大概的 attack plan(根据自己的初始知识)。然后根据这个计划，尝试解决问题。在解决问题的过程中，再不断的解决技术的细节。

一个好的 attack plan 需要设计者有好的全局观。对于一个领域的全貌有着比较深刻的理解和认识。这样，在设计中才能够知道什么能够做，什么不能做。

由此可见，做好一个好的学习笔记，很不容易。因为做学习笔记，不仅仅要考验一个人的知识存储水平，还考验一个人的思维方式。我们应当多学习好的思维方式，多积累，然后逐渐的增强自己这个方面的能力。

如何管理知识

知识的管理，能够通过软件来很好的辅助。比如，可以使用印象笔记来帮助自己管理知识。
印象笔记的定位很清楚，就是第二大脑。在信息的相关搜索上，印象笔记无疑是最好用的软件。不过，我更人更加倾向把自己已经整理过的，或者看到的文章放在印象笔记中。也就是，放入印象笔记的东西，要不就是已经理解的知识的摘要，要不就是之后“一定会看”的资料。

而平日里做读书笔记，个人感觉 margin note 可能是更好的选择。Margin note 的设计，和强力研读的概念比较接近。能够在书上做碎片化的笔记，并且将这些笔记做成 Mindnode 的形式。

而课堂和会议笔记，one note 或者 explain everthing 是第一选择。这两款软件对于 apple pencil 都有着非常好的支持。

另外，不管软件多方便，多好用，最终的目标都是为了帮助自己更好的理解，更好的思考。千万不要成为软件的奴隶，而是要成为软件的主人。这也是为什么我们需要结合那么多款学习笔记的软件的原因，因为目前没有一个 all in one 的解决方案，因此我们是从不同的软件中 extract 自己需要的功能，并且把每个软件发挥到极致。

如何更好的思考

方法和工具

一般而言一个完整的思考流程大概有下面三个最重要的元素为:

  1. 信息输入
  2. 信息整合
  3. 新的信息输出

信息的输入，是我们当前能够想到的相关联的知识。在这个方面，一个有效工具是思维导图。根据万老师的说法

人的短期工作记忆 —— 相当于大脑的“内存” —— 非常有限，每时每刻只能最多想四个东西。思维导图的正确用法，就是给大脑扩充内存。你只能想四个东西，可是做这件事需要同时考虑八个东西，那么最好的办法就是把这八个东西都写下来，摆在你面前帮助思考。

因此，我们在思考的时候，可以采用思维导图这个工具。而且好的思维导图，一定是很混乱的，而不是很漂亮的。思维导图应当是我们思考的 “highlight” 和 “connection”。

思维导图直观的反映了自己的想法观点和它们之间的连接。不过思维导图要用好，还需要对产生连接的想法做一个清晰的表达。比如，在两个连接的观点的线上，要记录这个连接存在的原因(assumption)。不要随意的把两个观点连接在一起，很多时候这种随意的连接会导致低质量的思考。
